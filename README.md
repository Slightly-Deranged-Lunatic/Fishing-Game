- New evironments / biomes
- More fish and rarities

# Installation
## Prequestites
1. Python

    1.1 You can check if you have python installed by running ``python --version`` in cmd or any terminal
    
     If you get something like ``Python 3.14.2`` then you have it installed and you should be good to go.
    
    1.11 If you don't get something like that then you can download python from [here](https://www.python.org/downloads/)

## Actual installation
1. Download the [latest release](https://github.com/Slightly-Deranged-Lunatic/Fishing-Game/releases) as a zip file

2. Unzip the zip file in a directory of your choosing

3. Run main.py in the folder and you're good to go!

    3.1 This can be done by right clicking on the file
